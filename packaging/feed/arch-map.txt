# WayHop feed arch map — one line per Go build:  <goname>  <openwrt tokens...>  |  <entware token>
#
# A portable static Go binary is re-stamped into one package per OpenWrt CPU-FAMILY token: a device
# only ACCEPTS a package whose Architecture is in its own list (`opkg print-architecture` / the apk
# arch). So from each Go binary we emit several tiny packages that differ ONLY in that field. Add
# subtargets here as you meet them; exotic ones use the documented --force-architecture escape hatch.
#
